Placeholders for character images.

from `index.html`. Recommended size: square 512x512 or 640x640 PNGs.

Examples referenced in the grid:
- ryu.png
- ken.png
- chunli.png
- guile.png
- blanka.png

If an image is missing, the UI will fall back to `placeholder.png`.
